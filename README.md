# This repository contains the source and configuration for my personal web presence.

- **ramonbuzo.tech** – My main personal website, used as a central hub for my online presence, projects, and professional information.
- **me.ramonbuzo.tech** – A personal profile/links page that aggregates important links, profiles, and resources related to me across the web.

Together, these sites act as the public entry point to my work, identity, and projects online.


## Content change — only sync files

## New service or change in docker-compose.yml
```git commit -m "feat: add redis container [deploy:restart]"```

## Full reset — delete volumes and recreate everything (use with caution)
```git commit -m "fix: clean DB and rebuild [deploy:reset]"```
